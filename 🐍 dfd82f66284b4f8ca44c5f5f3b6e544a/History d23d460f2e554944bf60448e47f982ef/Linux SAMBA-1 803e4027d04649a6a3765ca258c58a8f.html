SMB/CIFS : 윈도우와 리눅스가 공유할수 있게한다

윈도우에서 C://samba(공유할 디렉토리) 생성

위와 같이 설정

cmd를 관리자모드로 켠다

User(root) 추가

net user /add root 1234

net user

Centos 1로 접속

필요항목 확인

[root@localhost ~]# rpm -qa | grep samba
samba-client-libs-4.10.16-5.el7.x86_64
samba-libs-4.10.16-5.el7.x86_64
samba-common-libs-4.10.16-5.el7.x86_64
samba-client-4.10.16-5.el7.x86_64
samba-common-4.10.16-5.el7.noarch
[root@localhost ~]# yum -y install cifs-utils-devel.s

공유파일 확인

[root@localhost ~]# smbclient -L 10.0.2.6
Enter SAMBA\root's password: 

	Sharename       Type      Comment
	---------       ----      -------
	ADMIN$          Disk      Remote Admin
	C$              Disk      Default share
	IPC$            IPC       Remote IPC
	samba           Disk      
Reconnecting with SMB1 for workgroup listing.
do_connect: Connection to 10.0.2.6 failed (Error NT_STATUS_RESOURCE_NAME_NOT_FOUND)
Unable to connect with SMB1 -- no workgroup available

mount 작업

[root@localhost ~]# mkdir /samba1
[root@localhost ~]# mount -t cifs //10.0.2.6/samba /samba1

확인을 위해 윈도우에서 test.txt추가

[root@localhost ~]# ls /samba1/
test.txt

확인

window 2017 이후에는 뒤에 -o vers=2.0 붙여야한다

mount -t cifs //10.0.2.6/samba /samba1 -o vers=2.0

로그인없이 공유파일 접속하기

초기화

umount /samba1

유저 정보 등록

vi /etc/cred

username=root
password=1234

mount

mount -t cifs -o credentials=/etc/cred //10.0.2.17/samba /samba1

autofs를 이용한 자동마운트

직접 맵 마운트

초기화 : umount /samba1

autofs 설치 : yum install -y autofs

마스터 맵 설정

vi /etc/auto.master.d/direct.autofs

/-      /etc/auto.direct

직접 맵 설정

vi /etc/auto.direct


/samba1 -fstype=cifs,credentials=/etc/cred      ://10.0.2.6/samba

설정 적용 및 확인

[root@localhost ~]# systemctl restart autofs
[root@localhost ~]# ls /samba1
test.txt

간접 맵 마운트

초기화

[root@localhost ~]# systemctl stop autofs
[root@localhost ~]# umount /samba1
umount: /samba1: not mounted
[root@localhost ~]# rm -rf /etc/auto.master.d/direct.autofs

vi /etc/auto.master.d/indirect.autofs

/indirect       /etc/auto.indirect

vi /etc/auto.indirect

share   -fstype=cifs,credentials=/etc/cred      ://10.0.2.6/samba

디렉토리를 만든 후 설정 적용 후 확인

[root@localhost ~]# mkdir /indirect
[root@localhost ~]# mkdir /indirect/share
[root@localhost ~]# systemctl restart autofs
[root@localhost ~]# ls /indirect/share
test.txt
