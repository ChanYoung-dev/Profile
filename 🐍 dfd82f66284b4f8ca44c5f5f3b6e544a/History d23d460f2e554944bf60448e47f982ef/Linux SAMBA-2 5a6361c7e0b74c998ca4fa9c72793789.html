samba를 이용한 리눅스서버

서버구축

samba 설치 : yum -y install samba

samba서비스 실행

설치는 samba 써비스이름은 smb

systemctl restart smb

systemctl enable smb

공유 디렉토리 생성 및 권한 부여

[root@localhost ~]# mkdir /samba
[root@localhost ~]# chmod 777 /samba
[root@localhost ~]# touch /samba/everbody

samba설정 파일 수정

samba설정파일 백업

cp /etc/samba/smb.conf /etc/samba/smb.conf.bak

samba설정파일 수정

vi /etc/samba/smb.conf

[global]
        #workgroup = SAMBA
        workgroup = workgroup # 윈도우에서 workgroup설정 이름
        security = user # user일시 로그인 비밀번호필요
...
...
# 추가할 것
[sambatest]
        comment = Hi its samba comment
        path = /samba
        read only = no
        valid users = test

문법 오류 체크

testparm /etc/samba/smb.conf

smb계정 추가

smbpasswd -a test

설정 적용

[root@localhost ~]# firewall-cmd --add-service=samba --permanent
success
[root@localhost ~]# firewall-cmd --add-service=samba-client --permanent
success
[root@localhost ~]# firewall-cmd --reload
success
