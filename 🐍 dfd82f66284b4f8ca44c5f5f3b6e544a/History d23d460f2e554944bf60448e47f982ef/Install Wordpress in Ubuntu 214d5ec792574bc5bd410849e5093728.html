Ubuntu 에서 워드프레스 설치

apache 웹서버 설치

sudo apt update sudo apt install apache2 -y sudo systemctl stop apache2.service sudo systemctl start apache2.service sudo systemctl enable apache2.service

클라이언트에서 IP 접속 테스트

Mariadb(mysql) 설치 및 활성화

sudo apt-get install mariadb-server mariadb-client

  • 이거 안되면 밑에서 리포지토리 추가 하고 그 이후 하면 된다.

sudo systemctl stop mariadb.service sudo systemctl start mariadb.service sudo systemctl enable mariadb.service

mysql 보안설정

sudo mysql_secure_installation

Enter current password for root (enter for none): Just press the Enter
Set root password? [Y/n]: Y
New password: Enter password
Re-enter new password: Repeat password
Remove anonymous users? [Y/n]: Y
Disallow root login remotely? [Y/n]: Y
Remove test database and access to it? [Y/n]:  Y
Reload privilege tables now? [Y/n]:  Y

mysql 접속

sudo mysql -u root -p

php설치를 위한 레포지스토리 변경

sudo apt-get install software-properties-common
sudo add-apt-repository ppa:ondrej/ph

php7.2 설치

sudo apt update
sudo apt install php7.2 libapache2-mod-php7.2 php7.2-common php7.2-mysql php7.2-gmp php7.2-curl php7.2-intl php7.2-mbstring php7.2-xmlrpc php7.2-gd php7.2-xml php7.2-cli php7.2-zip

php 동작 확인

sudo nano /var/www/html/phpinfo.php


<?php phpinfo( ); ?>

접속해서 php 동작 확인 끝났으면 phpinfo.php 지우기

DATABASE User 생성

sudo mysql -u root -p : 데이터베이스 접속

CREATE DATABASE wpdatabase;
CREATE USER 'wpuser'@'localhost' IDENTIFIED BY '1234';
GRANT ALL ON wpdatabase.* TO 'wpuser'@'localhost' IDENTIFIED BY '1234' WITH GRANT OPTION;
FLUSH PRIVILEGES;
EXIT;

워드프레스 다운

cd /tmp
wget https://wordpress.org/latest.tar.gz    // 워드프레스 최신 버전을 다운로드
tar -xzf latest.tar.gz                      // 압축을 푼다.

web resource 자료로 워드프레스자료 이동

sudo mv /tmp/wordpress/* /var/www/html/

web resource 권한부여

sudo chown -R www-data:www-data /var/www/html/
sudo chmod -R 755 /var/www/html/

사이트 접속시 index.html이 뜨는 것을 방지

cd /var/www/html/

sudo rm -rf index.html

Apache 웹서버 재시작

sudo systemctl restart apache2.service

http://10.0.2.8 접속 후 설정

http://10.0.2.8/wp-admin/ 에 접속
